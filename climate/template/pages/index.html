  • Web Semántica

    La web semántica está constituida por 2 grandes visiones cuya confluencia suele crear confusión. La primera visión extraída de la Inteligencia artificial dice que la web semántica son iniciativas destinadas a promover una futura web cuyas páginas están organizadas codificadas y estructuradas de forma que los ordenadores puedan realizar razonamiento e inferencias a partir de la información con que se cuenta. La segunda visión que habla sobre el procesamiento robusto, dicta que la web semántica es el esfuerzo de convertir toda la World Wide Web en una enorme base de datos capaz de soportar procesamiento sistemático y consistente de la información.
